Effortless Image and Caption Generation

Inspired by Jambot, Image Jam uses stickies and images on canvas so you can focus in one view and keep track of your exploration.

Use Cases

Generate image with text: Create an image by connecting a sticky to Image Jam as a text prompt. Since it's a widget, you can reuse any stickies, including those generated from Jambot to jumpstart your session.
Generate caption: Write a description for an image by connecting an image to Image Jam. The output is a sticky upon which you can feed to a Jambot or iterate for the next image prompt.
Refine image: Iterate on an image by connecting a sticky and an image to Image Jam. This is useful for iterating on images and keeping track of their changes along the way.
